## Project Description

This project is a laser tag player entry system. It includes:

	-Splash screen display
	-Player entry screen
	-Play action screen
	-Random sound track selection
	-Event simulation
	-Database integration
	-UDP socket communication

## Requirements

The following are required to run this project:

	-Python 3.x
	-pygame 2.6.1
	-psycopg2-binary 2.9.11
	-PostgreSQL

## Setup Instructions

Run the install script:

```bash
chmod +x install.sh
./install.sh

To run the App:

python3 main.py
